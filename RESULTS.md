## Honest caveats

1. **N2 simplified estimator is conservative**. The closed-form `ΔN_eff` used in
   the script is a one-loop thermal estimate; for c′=0.05 over ℓ ∈ [0.1, 10] μm
   it numerically exceeds the ACT DR6 3σ bound. The paper's compatibility claim
   rests on the full Boltzmann / freeze-in treatment, which is not re-run here.
   The sanity check still executes and the figure is produced.
2. **D5 is an Euler-characteristic scaffold**. The β_k proxy via χ(ν) is exact
   only in the simply-connected regime of the Gaussian sub-level sets and holds
   asymptotically as a Morse-theory count; a full β₀/β₁/β₂ persistence computation
   on an N-body snapshot (GUDHI) is still a TODO, as documented in the script.
3. The venv the owner specified (`.venv-compute`) was created but its interpreter
   was not reachable from the execution sandbox; all runs used the working
   `~/.openclaw/venvs/pipeline/` interpreter which carries identical package
   versions. If a strictly hermetic rerun is needed, `pip install -r` inside
   `.venv-compute` should reproduce identical outputs.
