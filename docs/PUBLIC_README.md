# ECI — public reader's intro

**Entanglement, Complexity, Information** (ECI) is a framework paper in
theoretical cosmology. It does not derive new physics from scratch; instead it
threads six independently established 2022–2026 research programs —
observer-dependent type-II von Neumann algebras via the QRF crossed product;
DESI DR2 dynamical-dark-energy preferred by non-minimally coupled thawing
quintessence; Early Dark Energy; the Dark Dimension scenario; Cryptographic
Censorship; and persistent-homology diagnostics of primordial
non-Gaussianity — into a single predictive architecture. The upshot is a small
number of falsifiable near-term tests (a PPN bound $|\xi_\chi| \lesssim 10^{-3}$
testable by DESI DR3 + Euclid, a dark-dimension fifth-force signature at
$c' \simeq 0.05$, and a persistent-homology signal in the primordial bispectrum
within reach of LiteBIRD / CMB-S4).

**Manuscript state.** The current public head is the v4.8-dev pre-release line
that feeds into the v5.0 submission to *European Physical Journal C*. v5.0 is
cut when the hi_class NMC MCMC run delivers the $\xi_\chi$ posterior — the
submission package (svjour3 manuscript, cover letter, suggested referees,
AI-use disclosure, author-status note) is staged under `submission/epjc/`.
Every derivation, numerical check, adversarial review, and decision note is
version-controlled in this repository; there is no hidden supplementary
material. The ground-truth summary for readers who want to know exactly what
is claimed and what is merely conjectured lives at
[`paper/_internal_rag/INDEX.md`](../paper/_internal_rag/INDEX.md).
