## 2. Araki Relative Modular Operator — Mathematical Setup

### 2.1 The Tomita-Takesaki-Araki construction

Let M be a von Neumann algebra acting on a Hilbert space H. Let sigma be a faithful normal state on M with cyclic separating GNS vector |sigma> in H.

For a second faithful normal state rho with GNS vector |rho> (in the natural positive cone P^sharp of M), the **Araki relative modular operator** Delta_{rho|sigma} is defined via the closure of the antilinear map.
The modular Hamiltonian is defined with the uniform convention K_R = -log Delta_sigma >= 0
(adopted throughout this portfolio; in the type-I toy, Delta_sigma = sigma in the GNS sense,
so both conventions K_R = -log sigma and K_R = -log Delta_sigma coincide):

```
S_{rho|sigma} : A|sigma> --> A*|rho>   (A in M)
```

The polar decomposition S_{rho|sigma} = J_{rho|sigma} Delta_{rho|sigma}^{1/2} gives:
- Delta_{rho|sigma} = S_{rho|sigma}* S_{rho|sigma} (the relative modular operator)
- J_{rho|sigma} = modular conjugation (an antiunitary)

**Araki relative entropy** (Araki 1976, Comm. Math. Phys. 66, 83):
```
S(rho||sigma) = -<rho| log Delta_{rho|sigma} |rho>
```

This formula is valid whenever rho is not orthogonal to sigma (i.e., supp(rho) subset supp(sigma)). If the support condition fails, S(rho||sigma) = +inf.

### 2.2 Applicability to type-II_inf

**Key point**: the Araki 1976 construction is algebraic and does not require M to be type-I or even semifinite. The relative modular operator Delta_{rho|sigma} is defined purely in terms of the GNS vectors and the Tomita-Takesaki theory.

For a **type-II_inf** algebra (such as the CLPW/DEHK SdS crossed-product), one works with normal semifinite faithful (nsf) *weights* rather than states. The standard reference is:

- **Pusz-Woronowicz 1978** (Lett. Math. Phys. 2, 505; note: the standard citation for their functional calculus paper is Lett. Math. Phys. 5, 85 (1978), but the relative entropy variational formula appears in their 1975 Rep. Math. Phys. 8, 159 paper and the 1978 Lett. Math. Phys. note).

  **Honest caveat**: The exact Pusz-Woronowicz (1978) reference "Lett. Math. Phys. 5, 79" cited in Week 2 brief cannot be fetched directly (pre-arXiv). The Pusz-Woronowicz variational formula for relative entropy is:

  ```
  S(rho||sigma) = sup_{0 < A in M} { rho(log A) - log sigma(e^A) }
  ```

  This variational characterisation extends to weights in type-II_inf semifinite algebras via the Haagerup trace tau_II, replacing rho(.) with tau_II(rho .) and sigma(.) with tau_II(sigma .). This is consistent with Pusz-Woronowicz's approach and is used in the mathematical physics literature (see e.g. Ohya-Petz, "Quantum Entropy and Its Use," Springer 1993, Ch. 5).

- For the DEHK type-II_inf algebra M_R with the canonical nsf weight phi_0 (the Haagerup trace), the Araki relative entropy between two KMS-compatible states extends to:

  ```
  S(rho||sigma) = tau_II( rho (log rho - log sigma) )
  ```

  provided rho and sigma are density-matrix-like elements in L^1(M_R, tau_II) with rho << sigma.

**Status on Obstruction 1**: The Araki/Pusz-Woronowicz approach **does** define S(rho||sigma) for type-II_inf weights rigorously, without requiring the Petz map. Obstruction 1 from Week 1 does NOT apply to the relative modular operator approach.

## 3. Differentiating S(rho || sigma_t) in Modular Time

### 3.1 Setup

Let:
- M_R = type-II_inf crossed-product algebra for the SdS cosmological observer (DEHK arXiv:2412.15502)
- sigma = KMS state at inverse temperature beta = 2pi (the algebraic TFD / Hawking equilibrium)
- tau_t = modular automorphism group of sigma, generated by the modular Hamiltonian K_R:
  ```
  tau_t(A) = e^{i t K_R} A e^{-i t K_R}   (in the GNS representation)
  ```
  **Convention note**: We adopt the uniform sign convention K_R = -log Delta_sigma >= 0
  (so that K_R is a non-negative operator and the modular flow is e^{i t K_R}).
  In the type-I toy, Delta_sigma = sigma in the GNS sense, so K_R = -log sigma,
  and both conventions coincide.
- sigma_t = sigma o tau_{-t} (the state evolved in modular time)
- rho = a perturbed state (e.g. Boulware vacuum, or state with additional matter)

We want: d/dt S(rho || sigma_t).

### 3.2 The derivative formula

For type-I (density matrices), the formula is standard (see e.g. Uhlmann 1977; Lindblad 1974):

```
d/dt S(rho || sigma_t)|_{t=0} = -rho(K_R) = -<K_R>_rho
```

where K_R = -log sigma is the modular Hamiltonian of sigma (Rindler/de Sitter wedge operator).

**For type-II_inf**: The formula extends via the following chain of arguments:

1. The relative entropy S(rho || sigma_t) is differentiable in t for KMS states under the modular flow (Araki 1977, Pub. RIMS 13, 173; Bratteli-Robinson Vol. 2, Thm. 5.3.15).

2. Since sigma_t = sigma o tau_{-t}, one has:
   ```
   log Delta_{rho|sigma_t} = log Delta_{rho|sigma} + t K_R   (in the GNS sense)
   ```
   (This follows from the cocycle identity for relative modular operators: Delta_{rho|sigma_t} = Delta_{rho|sigma} e^{t K_R}.)

3. Differentiating:
   ```
   d/dt S(rho||sigma_t) = d/dt [-<rho| log Delta_{rho|sigma_t} |rho>]
                        = -<rho| K_R |rho>
                        = -<K_R>_rho
   ```

**This formula is rigorous for type-II_inf** provided:
- K_R is affiliated with M_R (it is, by construction as the modular Hamiltonian in the DEHK/CLPW framework)
- rho is in the domain of K_R (requires rho to be sufficiently regular, e.g. in the domain of the modular flow)

**Status on Week 1 Obstruction 1 (revisited)**: The derivative formula holds in type-II_inf via Araki-Bratteli-Robinson without constructing a Petz map. The rigorous extension requires checking that <K_R>_rho is well-defined (finite) in the DEHK algebra — this is plausible for the Boulware/perturbed vacuum but has not been verified explicitly for the SdS case.

### 3.3 Connection to generalized entropy

From DEHK (arXiv:2412.15502), the generalized entropy S_gen is:

```
S_gen(rho) = tau_II(rho log rho) + A/(4 G_N hbar)   [modulo additive const from type-II trace]
```

The relative entropy S(rho || sigma) = S_gen(sigma) - S_gen(rho) + (boundary terms that vanish for KMS reference; see Faulkner-Speranza arXiv:2405.00847 derivation of GSL).

Therefore:
```
d/dt S_gen(rho)|_{\rho fixed, sigma evolving} = -d/dt S(rho||sigma_t) = +<K_R>_rho
```

**Interpretation**: The rate of change of S_gen along modular time equals the expectation of the modular Hamiltonian K_R in the state rho. This is consistent with the Faulkner-Speranza GSL derivation (arXiv:2405.00847 abstract: "reproduce Wall's result relating the GSL to monotonicity of relative entropy of the quantum field algebras").

## 4. Connecting <K_R>_rho to C_k via DSSYK

### 4.1 The HPS identification

Heller-Papalini-Schuhmann (arXiv:2412.17785, PRL 135, 151602) establish in the DSSYK / sine-dilaton gravity model:

```
2 |log q| C_k = L_hat   (at disk level, finite temperature)
```

where:
- C_k = Krylov spread complexity of the TFD state
- L_hat = renormalized bulk length (complexity = volume observable)
- q = DSSYK deformation parameter

The bulk length L_hat is dual to the bulk volume, which is (by complexity = volume) proportional to the action of the Wheeler-DeWitt patch.

**Critical structural gap**: In the DSSYK setting, the model is type-I (finite-dimensional Hilbert space). The modular Hamiltonian K_R in the DSSYK model acts on the chord Hilbert space, not on the type-II_inf crossed-product algebra of the SdS continuum.

### 4.2 The proposed connection chain

The proposed link is:

```
dS_gen/dtau_R = <K_R>_rho  [Araki formula, type-II_inf]
                           ||
                           ?
2|log q| dC_k/dtau_R = dL/dtau_R  [HPS, type-I DSSYK]
```

For the two sides to meet, one needs:
1. An identification K_R (continuum SdS modular Hamiltonian) ~ L_hat (DSSYK bulk length operator)
2. A map between the type-I DSSYK Hilbert space and the type-II_inf DEHK algebra

**Assessment (Obstruction 2-prime)**: This is the new obstruction for Week 2. The HPS formula 2|log q| C_k = L is a *holographic* formula at the disk level in DSSYK/sine-dilaton gravity. It is NOT a statement about the modular Hamiltonian of a crossed-product algebra. To use it in the present context, one would need:

- A proof that the DSSYK modular Hamiltonian (which generates the Krylov spread) equals the continuum K_R up to a constant.
- A type-II_inf analogue of the complexity = volume proposal for SdS (not established in CLPW or DEHK).

Neither exists in the current literature. This is a **new conceptual gap**, distinct from the three Week 1 obstructions.

### 4.3 What IS established

The formula dS_gen/dtau_R = <K_R>_rho is well-founded in type-II_inf. The modular Hamiltonian K_R is the same operator that appears in the first law of thermodynamics for the de Sitter horizon:

```
delta S_gen = beta_H delta <K_R>   (first law)
```

This is independent of any complexity conjecture. It gives a formula for dS_gen/dtau_R without reference to C_k.

## 5. Honest Assessment: Does Week 2 Overcome Week 1's Three Obstructions?

### Obstruction 1: No type-II_inf Petz construction

**STATUS: RESOLVED by Week 2 approach.**

The Araki relative modular operator Delta_{rho|sigma} is defined for type-II_inf weights. The Pusz-Woronowicz variational formula extends to the DEHK setting. No Petz map is needed.

**Caveat**: The rigorous statement requires checking that K_R is in the domain of the expectation <.>_rho for the relevant class of states (Boulware, perturbed thermal). This has not been done explicitly for SdS in the literature but is a well-posed mathematical question (not a no-go).

### Obstruction 2: Deficit does not factor through C_k

**STATUS: RECAST, NOT RESOLVED.**

In the Araki approach, the relevant quantity is <K_R>_rho, not the Petz deficit. The question becomes: does <K_R>_rho relate to C_k?

From HPS, C_k is the Krylov complexity in the chord Hilbert space of DSSYK. The modular Hamiltonian K_R for the SdS observer is a different operator in a different algebra. A functional relationship <K_R>_rho = g(C_k) would require:

- Identifying K_R with the DSSYK Krylov generator (not established)
- Or using 2|log q| C_k = L and identifying L with <K_R>_rho via CV (not established at the level of the DEHK type-II_inf algebra)

**New Obstruction 2' (Algebra type mismatch)**: The HPS result lives in a type-I (finite-dimensional) theory. The DEHK algebra is type-II_inf. The connection between C_k and <K_R>_rho requires bridging these two frameworks — a non-trivial open problem.

### Obstruction 3: Sign change at Page time

**STATUS: PARTIALLY RECAST.**

The formula dS_gen/dtau_R = <K_R>_rho does NOT resolve the sign issue. In fact:
- For tau_R < tau_Page (growing phase): <K_R>_rho > 0 (the state rho has higher modular energy than the KMS reference sigma)
- For tau_R > tau_Page (decreasing phase): <K_R>_rho < 0 (expected by Page curve thermodynamics)

This is now a statement about the sign of <K_R>_rho, which can in principle be tracked through the Page time. The Week 2 approach makes the sign issue **more explicit and tractable** (it is now a statement about a single observable rather than a complex functional), but does not eliminate it.

A lower bound of the form dS_gen/dtau_R >= f(C_k) with f >= 0 still fails on the decreasing branch unless one restricts to the growing phase or takes f = |f| (two-sided bound).

## 6. Summary Assessment and Verdict

### What Week 2 achieves

1. **Resolves Obstruction 1**: The Araki/Pusz-Woronowicz relative modular operator is rigorously defined for type-II_inf. The formula dS_gen/dtau_R = <K_R>_rho holds in this setting (up to domain regularity of K_R).

2. **Provides a cleaner formula**: dS_gen/dtau_R = <K_R>_rho is a first-principles result, derivable without Petz recovery and without DSSYK.

3. **Reframes but does not resolve the C_k connection**: The gap between <K_R>_rho (continuum SdS modular Hamiltonian) and C_k (DSSYK Krylov complexity) is now the central open problem.

4. **Makes the sign issue explicit**: The sign of <K_R>_rho tracks the growing/decreasing entropy phases.

### What Week 2 does NOT achieve

- A direct proof that dS_gen/dtau_R >= f(C_k) for any explicit f. The C_k connection requires bridging type-I DSSYK and type-II_inf SdS — a new obstruction (Obstruction 2').

- A resolution of the Page time sign issue for a universal lower bound.

### Verdict

**WEEK 2 PATH: PARTIALLY VIABLE — formula derived, C_k connection remains open**

More specifically:
- The formula dS_gen/dtau_R = <K_R>_rho is **derivable** and **rigorous** in Week 3 if K_R domain conditions are verified.
- The connection to C_k via HPS is **speculative** and requires a separate identification of K_R with the DSSYK Krylov generator.
- Prop. 1bis cannot be written as a theorem without this identification.

The Week 2 result is a **genuine theorem candidate** for the formula dS_gen/dtau_R = <K_R>_rho in type-II_inf, which is worth stating in v6.2 S2 independently of the C_k connection.
