# M63 Dispatch Instructions — SageMath Installation + Falsifier Dispatch

## Step 1: Install SageMath on Hostinger VPS

```bash
sudo apt update
sudo apt install -y sagemath sagemath-jupyter
```

Verify:

```bash
sage --version
# Expected: SageMath version 9.x or 10.x
```

Alternatively, use conda (faster, more up-to-date):

```bash
conda create -n sage python=3.11
conda activate sage
conda install -c conda-forge sage
```

## Step 2: Sanity check — Sage can see the newform

```bash
cd /root/crossed-cosmos/notes/eci_v7_aspiration/M63_SAGE_SCRIPTS/
sage -c "
from sage.all import Newforms, DirichletGroup, QQ
chi = DirichletGroup(4, QQ)[1]
S = Newforms(4, 5, character=chi, names='a')
f = S[0]
print('a_2 =', f.q_expansion(3)[2])
print('Expected: -4')
"
```

If this returns `a_2 = -4`, Sage is working and the newform is accessible.
