## §1. Setup: FRW Geometry, Observer, Field, and Vacuum

### 1.1 Metric and conformal coordinates

Fix **radiation-dominated FRW** in conformal time η with flat spatial sections (Cartesian):

```
ds²_FRW = a(η)²(−dη² + dx¹² + dx²² + dx³²),    a(η) = a₀ η,    η ∈ (0, ∞).
```

The **conformally related flat Minkowski metric** is:

```
ds²_M = −dη² + dx¹² + dx²² + dx³².
```

The two are related by `ds²_FRW = a(η)² ds²_M`. The conformal factor `Ω(η, x) = a(η) = a₀ η` depends only on conformal time and vanishes at `η = 0` (the Big Bang singularity).

**Cosmic proper time** along a comoving worldline `x = 0`:

```
τ = ∫₀^η a(η') dη' = (a₀/2) η²,    ⟹    η(τ) = (2τ/a₀)^{1/2}.
```

The scale factor in cosmic time: `a(τ) = a₀ η(τ) = (2a₀ τ)^{1/2} ∝ τ^{1/2}`, the standard radiation-dominated power law.

The **conformal Hubble rate** (also called the conformal weight function):

```
ω_η(η) := a'(η)/a(η) = 1/η.
```

In cosmic time: `ω(τ) := (d/dτ) log a = (da/dτ)/a = 1/(2τ)`. Note `ω → ∞` as τ → 0 (near Big Bang).

### 1.2 Comoving observer and causal diamond

The observer follows the worldline `x = 0`, `η ∈ (η_i, η_f)` with `0 < η_i < η_f < ∞`. We ALWAYS assume `η_i > 0`, meaning the observer's causal domain does **not** reach the Big Bang singularity. This assumption is crucial for all that follows and is relaxed only in §5.

The **causal diamond** in conformal coordinates:

```
D(O) = {(η, x) : |x| < η_f − η,  η > η_i + |x|}.
```

In the conformally related Minkowski frame, this is the identical set of coordinate points — the conformal map is the identity on coordinates `(η, x)` and acts only by rescaling the metric.

### 1.3 Conformally coupled massless scalar

The field of interest is the **conformally coupled massless scalar** in 3+1 dimensions. Its action is:

```
S[φ] = −½ ∫ d⁴x √{−g} [(∇φ)² + (1/6) R φ²],
```

with `ξ = 1/6` the conformal coupling constant for `d = 4`. The field equation in the FRW background is:

```
(□_{FRW} + R_{FRW}/6) φ = 0.
```

The **rescaled field** `φ̃ := a(η) φ` satisfies the flat Minkowski wave equation:

```
□_M φ̃ = 0.
```

This follows from the standard conformal transformation of the Laplace-Beltrami operator in `d=4`. The rescaling `φ ↦ φ̃ = a(η) φ` is the key unitary map between the two theories.

### 1.4 Vacuum states and their equivalence

The **Bunch-Davies vacuum** (or "conformal vacuum") for the conformally coupled massless scalar in radiation-dominated FRW is the state whose two-point function, in the conformal frame, coincides with the Minkowski Wightman function. More precisely, the GNS vacuum `Ω_{FRW}` for the Bunch-Davies state on `A(O)_FRW` is mapped to the **Minkowski vacuum** `Ω_M` on `A(M_O)_M` by the unitary `U` that implements `φ ↦ a(η) φ`. This is a standard consequence of the conformal invariance of the conformally coupled massless field equation (see Wald, *QFT in Curved Spacetime*, §4.6; Birrell-Davies, §6.7).

## §2. Conformal Isometry to the Minkowski Diamond

### 2.1 The unitary U

Define the Hilbert space map `U : H_{FRW} → H_M` by its action on (Cauchy data of) field operators:

```
U φ_{FRW}(η, x) U⁻¹ = a(η)⁻¹ φ̃_M(η, x),    i.e., φ̃_M = a(η) φ_{FRW}.
```

On test functions `f ∈ C₀^∞(D(O))`, this reads:

```
U φ_{FRW}(f) U⁻¹ = φ_M(a · f),    (a · f)(η, x) := a(η) f(η, x).
```

**Claim:** `U` extends to a **unitary operator** `H_{FRW} → H_M` that maps

```
U : A(O)_{FRW} → A(M_O)_M
```

as a **W*-algebraic isomorphism**, and satisfies `U Ω_{FRW} = Ω_M`.

*Argument for the claim:* The symplectic form for the conformally coupled scalar transforms as `σ_{FRW}(f, g) = σ_M(a·f, a·g)` (the `a²` from the metric volume element is cancelled by the `a⁻¹` from the two field redefinitions). Hence `U` is a symplectomorphism on classical solution spaces, extends to a unitary on Fock space, and maps the algebra generated by `{φ_{FRW}(f) : f ∈ C₀^∞(D(O))}` onto `{φ_M(h) : h ∈ C₀^∞(M_O)}`. Vacuum equivalence is by construction (Bunch-Davies = conformal vacuum). Since `D(O)` and `M_O` are the same set of coordinates, the causal structure — and hence the Haag-Kastler locality — is preserved.

This establishes: **the von Neumann algebras `A(O)_{FRW}` and `A(M_O)_M` are unitarily equivalent W*-algebras**.

### 2.2 The Bisognano-Wichmann modular flow in Minkowski

For the Minkowski algebra `A(M_O)` and the Minkowski vacuum, the Bisognano-Wichmann (BW) theorem applies when `M_O` is a **Rindler wedge** `W_R = {x¹ > |η|}`. For the causal diamond `D(O)` (doubly bounded region), the relevant theorem is the **Hislop-Longo theorem** (1982), which states that the modular automorphism group of `(A(D), Ω_M)` is implemented by a one-parameter group of conformal transformations of the diamond.

For concreteness we first treat the Rindler wedge case (all structural conclusions carry over to the diamond with the substitution of the Möbius flow for the boost). In `W_R`:

```
σ_s^{BW}(φ_M(η, x¹, x_⊥)) = φ_M(η cosh(2πs) + x¹ sinh(2πs),  x¹ cosh(2πs) + η sinh(2πs),  x_⊥).
```

The modular Hamiltonian `K_{BW} = -log Δ_{BW}` (where `Δ_{BW}` is the Tomita-Takesaki modular operator of `(A(W_R), Ω_M)`) is, by the BW theorem:

```
K_{BW} = 2π ∫_{x¹>0} d³x  x¹ T_{00}^M(η=0, x¹, x_⊥),
```

where `T_{00}^M = ½[(∂_η φ̃_M)² + (∇_x φ̃_M)²] + improvement terms` is the Minkowski stress-energy density for the massless scalar. (The improvement term for conformal coupling is `−(1/6) ∂_μ(φ̃ ∂^μ φ̃ − φ̃ □φ̃ g_μ⁰)` which vanishes on-shell for `□φ̃ = 0`.)

## §3. Pullback of the BW Modular Flow to FRW

### 3.1 The pulled-back modular operator

Since `U` implements a W*-isomorphism `A(O)_{FRW} ≅ A(M_O)_M` and maps `Ω_{FRW} ↦ Ω_M`, the **Tomita-Takesaki modular operator** transforms as:

```
Δ_{FRW} = U⁻¹ Δ_M U.
```

Consequently, the **FRW modular Hamiltonian** is:

```
K_{FRW} := −log Δ_{FRW} = −log(U⁻¹ Δ_M U) = U⁻¹ (−log Δ_M) U = U⁻¹ K_{BW} U.
```

This is the **pullback formula**: *K_{FRW} is the operator obtained by conjugating K_{BW} with the conformal rescaling unitary U*.

The modular automorphism group of `(A(O)_{FRW}, Ω_{FRW})` is:

```
σ_s^{FRW}(A) = e^{is K_{FRW}} A e^{-is K_{FRW}} = U⁻¹ [e^{is K_{BW}} (U A U⁻¹) e^{-is K_{BW}}] U
             = U⁻¹ σ_s^{BW}(U A U⁻¹) U.
```

**Explicit formula on field operators:** For `A = φ_{FRW}(η, x¹, x_⊥) ∈ A(O)_{FRW}`:

```
σ_s^{FRW}(φ_{FRW}(η, x)) = U⁻¹ σ_s^{BW}(U φ_{FRW}(η,x) U⁻¹) U
                           = U⁻¹ σ_s^{BW}(a(η)⁻¹ φ̃_M(η,x)) U
                           = a(η)⁻¹ · U⁻¹ σ_s^{BW}(φ̃_M(η,x)) U.
```

Now apply the BW boost to `φ̃_M`:

```
σ_s^{BW}(φ̃_M(η, x¹, x_⊥)) = φ̃_M(η(s), x¹(s), x_⊥)
```

where `(η(s), x¹(s))` is the BW boost orbit: `η(s) = η cosh(2πs) + x¹ sinh(2πs)`, `x¹(s) = η sinh(2πs) + x¹ cosh(2πs)`. Then:

```
U⁻¹ φ̃_M(η(s), x¹(s), x_⊥) U = a(η(s))⁻¹ · a(η(s)) φ_{FRW}(η(s), x¹(s), x_⊥)
                                 = φ_{FRW}(η(s), x¹(s), x_⊥).
```

Combining:

```
σ_s^{FRW}(φ_{FRW}(η, x¹, x_⊥)) = a(η)⁻¹ · a(η(s)) · φ_{FRW}(η(s), x¹(s), x_⊥)
                                  = (a(η(s))/a(η)) · φ_{FRW}(η(s), x¹(s), x_⊥).
```

**This is the explicit pullback formula.** The modular flow on field operators in FRW is the BW Lorentz boost on the coordinate pair `(η, x¹)`, dressed by the **conformal weight factor** `a(η(s))/a(η)`.

For `a(η) = a₀ η`, this weight is:

```
a(η(s))/a(η) = η(s)/η = [η cosh(2πs) + x¹ sinh(2πs)] / η.
```

On the t=0 surface (the wedge tip, relevant for BW), `η = 0` is singular — but as noted in §1.2, the observer's diamond has `η ≥ η_i > 0`, so `η` is bounded away from zero on all relevant field operator evaluations.

## §4. Computation of δK_{conf}

### 4.1 Formal splitting

Write:

```
K_{FRW} = U⁻¹ K_{BW} U = K_{BW} + δK_{conf},
```

where:

```
δK_{conf} := U⁻¹ K_{BW} U − K_{BW} = [U⁻¹, K_{BW}] U.
```

The commutator `[U⁻¹, K_{BW}]` captures how the conformal rescaling fails to commute with the modular Hamiltonian. To compute it explicitly, use `K_{BW} = 2π ∫ x¹ T_{00}^M d³x` and `U⁻¹ φ̃_M(x) U = a(η(x))⁻¹ a(η(x)) φ_{FRW}(x) = φ_{FRW}(x)`.

Conjugating `T_{00}^M` under `U⁻¹ (\cdot) U`: replace `φ̃_M → a(η) φ_{FRW}` everywhere.

```
T_{00}^M = ½ (∂_η φ̃)² + ½ (∇_x φ̃)²

U⁻¹ T_{00}^M(η, x) U = ½ (∂_η[a(η) φ_{FRW}])² + ½ a(η)² (∇_x φ_{FRW})²

= ½ [a'(η) φ_{FRW} + a(η) ∂_η φ_{FRW}]² + ½ a(η)² (∇_x φ_{FRW})²

= ½ a(η)² [(∂_η φ_{FRW})² + (∇_x φ_{FRW})²]
  + a(η) a'(η) φ_{FRW} (∂_η φ_{FRW})
  + ½ [a'(η)]² φ_{FRW}².
```

Therefore:

```
K_{FRW} = U⁻¹ K_{BW} U
         = 2π ∫_{x¹>0} d³x  x¹ {
             ½ a(η)² [(∂_η φ)² + (∇_x φ)²]          [Term I: rescaled kinetic]
           + a(η) a'(η) φ (∂_η φ)                     [Term II: cross term]
           + ½ [a'(η)]² φ²                             [Term III: mass-like]
         },
```

evaluated on the appropriate Cauchy surface (suppressing the Cauchy-surface index on `η`).

The **BW modular Hamiltonian expressed in φ_{FRW}** is:

```
K_{BW} = 2π ∫_{x¹>0} d³x  x¹ T_{00}^M(η,x)|_{φ̃ = a φ_{FRW}}
        = 2π ∫ x¹ [½ a²(∂_η φ)² + ½ a²(∇φ)² + a a' φ ∂_η φ + ½(a')² φ²] d³x.
```

Wait — this is exactly `U⁻¹ K_{BW} U = K_{FRW}`. The splitting `K_{BW} + δK_{conf}` must be understood differently: **K_{BW} acts on H_M** (on φ̃-operators), while **K_{FRW} acts on H_{FRW}** (on φ-operators). They act on different (though isomorphic) Hilbert spaces.

### 4.2 The correct interpretation of δK_{conf}

The perturbation is most meaningfully expressed by comparing the FRW modular Hamiltonian to the "naively expected" BW form written in FRW variables. Define the **canonical FRW modular Hamiltonian candidate** (what one would write if the BW boost were a geometric symmetry of the FRW spacetime):

```
K_{BW,FRW}^{naive} := 2π ∫_{x¹>0} d³x  x¹  T_{00}^{FRW}(η, x),
```

where `T_{00}^{FRW}` is the stress-energy density of `φ_{FRW}` in the FRW metric. For the conformally coupled scalar:

```
T_{00}^{FRW} = a(η)^{-4} [a(η)^2 T_{00}^M]_{φ̃=aφ}    [by conformal scaling of T_μν in d=4]
              = a(η)^{-2} T_{00}^M[aφ]
              = ½ a(η)^{-2} [a²(∂_η φ)² + a²(∇φ)²] + ...
              = ½ [(∂_η φ)² + (∇φ)²] + O(a'/a).
```

(Here the `−2` conformal scaling of the energy density in 4d from `g^{00} = a^{-2}` and `√{-g} = a^4`.)

So the actual FRW modular Hamiltonian is:

```
K_{FRW} = U⁻¹ K_{BW} U
         = 2π ∫ x¹ {½ a² [(∂_η φ)² + (∇φ)²] + a a' φ ∂_η φ + ½(a')² φ²} d³x.
```

The correction relative to a "BW-like" term `2π ∫ x¹ · a² T_{00}^{FRW}` (which would give `2π ∫ x¹ · ½[(∂φ)² + (∇φ)²] d³x · a²`…) is:

```
δK_{conf} = 2π ∫_{x¹>0} d³x  x¹ {a(η) a'(η) φ ∂_η φ + ½ [a'(η)]² φ²}.
```

For `a(η) = a₀ η`:

```
a'(η) = a₀,    a(η) a'(η) = a₀² η,    [a'(η)]² = a₀².
```

So:

```
δK_{conf} = 2π a₀ ∫_{x¹>0} d³x  x¹ {a₀ η · φ_{FRW} (∂_η φ_{FRW}) + ½ a₀ φ_{FRW}²}.
```

The dominant term as `η → 0` (i.e., near the Big Bang, or at early conformal times) is:

```
δK_{conf} ≈ π a₀² ∫_{x¹>0} d³x  x¹  φ_{FRW}²     (as η → 0).
```

This is a **mass-like term** — a quadratic in the field without derivatives.

## §5. Boundedness Analysis: The IR Obstruction

### 5.1 Is δK_{conf} bounded?

The operator `∫_{x¹>0} d³x  x¹  φ_{FRW}(x)²` is **not bounded** on the Fock Hilbert space `H_{FRW}`. In momentum space, the field two-point function at zero mass gives:

```
φ(x)² → ∫ d³k d³k'/(4π² |k||k'|)^{1/2} a†(k) a†(k') + h.c. + a†(k) a(k') + ...
```

and the expectation value of `∫ x¹ φ²` in one-particle states diverges as `|k| → 0` (infrared divergence: the weight `|k|⁻¹` is not integrable in 3d for the zero-mode). This is an operator-valued distribution that cannot be bounded.

### 5.2 Is δK_{conf} K_{BW}-bounded?

The standard relative-boundedness test: is `||δK_{conf} ψ|| ≤ α ||K_{BW} ψ|| + β ||ψ||` for some `α < 1`, `β < ∞`?

The K_{BW}-operator controls `∫ x¹ T_{00}^M = ∫ x¹ [(∂φ̃)² + (∇φ̃)²]`, i.e., the **derivative** content of the field. The term `∫ x¹ φ²` in `δK_{conf}` contains **no derivatives**. In the massless limit, there is no control of the field amplitude by its gradient in infinite volume:

```
||∫ x¹ φ² d³x ψ|| is NOT controlled by ||∫ x¹ (∂φ)² d³x ψ||.
```

Explicitly, in momentum space: `K_{BW}` acts roughly as multiplication by `|k|` (the Rindler energy), while `δK_{conf}` acts as `1` (a mass term). Their ratio `|k|⁻¹ → ∞` as `|k| → 0`. **Therefore `δK_{conf}` is NOT K_{BW}-bounded in any operator-theoretic sense.**

### 5.3 Why this does NOT kill type II_∞

This is the key point. The "bounded perturbation" test proposed in the previous note (`frw_non_stationary_attack_2026_05_02.md`, §3 Sub-result (b)) was **the wrong criterion**. Boundedness of `δK_{conf}` is **sufficient** for type II_∞ to survive, but it is NOT **necessary**.

The correct criterion is the **Connes spectrum** `Γ(σ)`, defined as the intersection over all non-zero σ-invariant projections of the Arveson spectrum. The fundamental observation is:

**Lemma (Connes spectrum under unitary equivalence):** Let `(M, σ)` and `(M', σ')` be W*-dynamical systems. If `α : M → M'` is a W*-isomorphism such that `σ'_s = α ∘ σ_s ∘ α⁻¹`, then `Γ(σ) = Γ(σ')`.

*Proof:* The Connes spectrum is a W*-algebraic invariant of the pair `(M, σ)`, not of its Hilbert space representation. Unitary equivalence preserves all such invariants. □

**Application:** The FRW modular automorphism `σ_s^{FRW}` and the BW modular automorphism `σ_s^{BW}` satisfy exactly this relation with `α = U⁻¹ (\cdot) U`:

```
σ_s^{FRW}(A) = U⁻¹ σ_s^{BW}(U A U⁻¹) U    for all A ∈ A(O)_{FRW}.
```

Therefore:

```
Γ(σ^{FRW}) = Γ(σ^{BW}).
```

The Bisognano-Wichmann modular automorphism for the Minkowski vacuum on a Rindler wedge (or Hislop-Longo for a diamond) is the generator of a type III_1 factor: its Connes spectrum is `Γ(σ^{BW}) = ℝ`. Hence `Γ(σ^{FRW}) = ℝ`, and by Connes-Takesaki classification theory:

```
M = A(O)_{FRW} ⋊_{σ^{FRW}} ℝ  is type II_∞.
```

### 5.4 The role of η_i > 0 (the IR/UV conditions for the isomorphism)

The argument in §5.3 requires that `U` implements a genuine W*-isomorphism of von Neumann algebras `A(O)_{FRW} ≅ A(M_O)_M`. This holds when:

(a) `D(O)` is a bounded open diamond with `η_i > 0` (so the conformal factor `a(η) = a₀ η` is bounded above and below on `D(O)`; specifically `a_{\min} := a₀ η_i > 0`).  
(b) The test function space `C_0^\infty(D(O))` is mapped bijectively to `C_0^\infty(M_O)` by `f \mapsto a \cdot f` (valid iff `a` is smooth and nonvanishing on `D(O)` — condition (a) guarantees this).  
(c) No anomalous boundary conditions arise at `∂ D(O)` that break the isomorphism — this is guaranteed by the Haag-Kastler axioms and locality.

**When η_i = 0 (observer's diamond touches the Big Bang):** Condition (a) fails because `a(η_i) = 0`. The rescaling `f \mapsto a_0 η \cdot f` sends functions with support near `η = 0` to functions that vanish there — this is NOT a bijection on test function spaces, and the W*-isomorphism breaks. In this limiting case the algebra `A(O)_{FRW}` for a diamond touching `η = 0` is not unitarily equivalent to `A(M_O)_M` via `U`, and the argument fails. The type of the crossed product in this case is **not determined** by the current approach.

## §6. Tentative Result and Open Questions

### 6.1 Main result (provisional)

**Theorem (informal):** Let `φ` be the conformally coupled massless scalar field in radiation-dominated FRW spacetime with `a(η) = a₀ η`. Let `D(O)` be a causal diamond for a comoving observer with `η_i > 0` (away from the Big Bang), and let `A(O)` be the associated von Neumann algebra in the GNS representation of the conformal (Bunch-Davies) vacuum `Ω`. Let `M = A(O) ⋊_{σ} ℝ` be the Connes-Takesaki crossed product by the modular automorphism group. Then:

```
M ≅ B(L²(ℝ))  (up to a tensor product with an abelian factor),
```

i.e., **M is type II_∞**. This does not require any Killing vector field on the FRW spacetime.

*Proof sketch:*
1. The conformal equivalence `(D(O)_{FRW}, ds²_{FRW}) ∼ (M_O, ds²_M)` for `a(η) = a₀ η` and the conformal coupling of `φ` imply the unitary map `U : H_{FRW} → H_M`.
2. For `η_i > 0`, `U` implements a W*-isomorphism `A(O)_{FRW} ≅ A(M_O)_M` (§2.1).
3. The Bunch-Davies vacuum maps to the Minkowski vacuum: `U Ω_{FRW} = Ω_M` (§1.4).
4. Therefore `σ^{FRW} = U⁻¹ ∘ σ^{BW} ∘ U` (§3.1), and `Γ(σ^{FRW}) = Γ(σ^{BW})` (§5.3).
5. The BW/Hislop-Longo modular flow generates a type III_1 factor `A(M_O)`, so `Γ(σ^{BW}) = ℝ`.
6. By Connes-Takesaki duality, `A(O) ⋊_σ ℝ` is type II_∞. □

**Headline:** *The crossed-product algebra for a comoving observer in radiation-dominated FRW (conformally coupled massless scalar, away from the Big Bang singularity) is type II_∞, establishing a new case of the gravitational algebra classification that requires no Killing vector.*

### 6.3 Open questions for the full paper

The following questions are not answered by the above argument and define the research agenda for subsequent months:

**Q1 (Massive scalar):** For `m > 0`, the field equation `(□_{FRW} + R/6 + m²) φ = 0` does NOT reduce to `□_M φ̃ = 0` under the conformal rescaling — an effective mass term `m² a(η)²` remains. The unitary `U` does NOT intertwine the field dynamics, and `A(O)_{FRW}^{m>0}` is NOT isomorphic to `A(M_O)_M^{m=0}`. The modular flow for massive fields in radiation-dominated FRW is unknown. The type of `A(O) ⋊_σ ℝ` for `m > 0` is an open problem.

**Q2 (Matter-dominated FRW):** For `a(η) = a₀ η²` (matter-dominated), the conformal coupling still gives `φ̃ = a(η) φ` as a massless Minkowski field, so the structural argument of §§2–5 applies. The conformal factor `a₀ η²` is still smooth and nonvanishing for `η > 0`. However, the conformal weight `ω(η) = a'/a = 2/η` diverges faster near `η = 0` than in the radiation case, but this does not affect the argument for `η_i > 0`. Expectation: the type II_∞ result extends to matter-dominated FRW. This should be verified explicitly.

**Q3 (Diamond touching η = 0):** When `η_i = 0`, the W*-isomorphism `U` breaks. Is `A(O)_{FRW}` still type III_1 for such diamonds? Or does the Big Bang singularity introduce a type III_0 or type II_1 factor? This is the hardest open question — it requires a direct analysis of the modular flow near the cosmological singularity, possibly using adiabatic regularization methods (Parker-Toms, Ch. 5).

**Q4 (Gravitational dressing and the clock):** The crossed-product construction `A(O) ⋊_σ ℝ` follows CLPW 2022 in including a gravitational clock degree of freedom `p_t` satisfying `[p_t, H] = iℏ`. The above argument establishes the type of `A(O) ⋊_σ ℝ` as an *abstract* crossed product by `Γ(σ) = ℝ`. The physical interpretation — that this represents the algebra of dressed gravitational observables — requires verifying that the clock coupling does not introduce additional structure that changes the type. This is expected to follow by the same CLPW argument, but should be checked.

**Q5 (Non-conformal scalars and interacting fields):** The argument is specific to `ξ = 1/6` (conformal coupling) and `m = 0`. For `ξ ≠ 1/6`, the rescaling `φ ↦ a φ` does not map the field equation to the flat equation. The modular flow for non-conformally-coupled fields in FRW is structurally different and should not be assumed to give type II_∞. Interacting fields (λφ⁴, etc.) break conformal invariance and are completely untreated.

**Q6 (Comparison with Gomez 2407.20671):** The Gomez paper (arXiv:2407.20671, verified) finds type III_1 factors for inflationary spacetimes viewed BEFORE the crossed-product step, consistent with our finding that `A(O)` is type III_1. Our result is about the type of the crossed product `M = A(O) ⋊_σ ℝ`, which is type II_∞. These are compatible. The paper should include a precise comparison establishing that no contradiction arises and clarifying the distinct role of the crossed-product step.

**Q7 (General FRW: criterion in terms of H(t)):** The Hubble parameter `H(t) = ȧ/a = ω(t)` encodes the conformal weight. The type II_∞ result holds whenever the conformal equivalence map `U` is a W*-isomorphism, which requires `a(η) > 0` on `D(O)`. Can the general condition be stated as: "*M is type II_∞ for any FRW background with `a(η) > 0` on `D(O)`, for the conformally coupled massless scalar*"? This would be a single theorem covering radiation, matter, and mixed epochs (away from singularities). The statement seems likely correct and should be formalized.

**Q8 (Uniqueness of the conformal vacuum):** The result assumes the Bunch-Davies vacuum = conformal vacuum = `U`-pullback of Minkowski vacuum. For excited states or mixed states (e.g., thermal states, squeezed states relevant to post-reheating FRW), the modular automorphism is state-dependent. The type of `A(O) ⋊_{σ_ρ} ℝ` for a general state `ρ` is not addressed and may differ.

## Appendix: δK_{conf} in Explicit Coordinates

For completeness, the correction term `δK_{conf}` expressed on the Cauchy surface `η = η_*` with `a(η_*) = a₀ η_*`, `a'(η_*) = a₀`:

```
δK_{conf} = 2π ∫_{x¹>0} d³x  x¹ {
    a₀² η_* · φ_{FRW} (∂_η φ_{FRW})          [cross term, odd in φ ↦ −φ]
  + ½ a₀² · φ_{FRW}²                          [mass-like term, positive semi-definite]
}.
```

The **cross term** is antisymmetric under `φ ↦ −φ` and has zero expectation value in any even state (including the vacuum). It is formally anti-self-adjoint but becomes self-adjoint when symmetrized. It represents a time-dependent rescaling effect.

The **mass-like term** `½ a₀² ∫ x¹ φ² d³x` is positive semi-definite, unbounded, and IR-divergent (as shown in §5.1). Its spectrum fills `(0, ∞)`, adding a positive shift to `K_{BW}`. This does NOT change `Γ(σ)` because the Connes spectrum is invariant under the unitary conjugation, regardless of how the perturbation looks in any fixed representation.

The key non-obvious point: **the unboundedness of `δK_{conf}` is not a problem** because `δK_{conf}` is not an independent perturbation — it is the *tautological* difference `U⁻¹ K_{BW} U − K_{BW}` arising entirely from representing the SAME operator in two different Hilbert spaces. The Connes spectrum of the *abstract* modular automorphism group is unaffected.
