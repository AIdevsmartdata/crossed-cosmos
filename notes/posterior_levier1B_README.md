**Honest caveats for the comparison:**
1. Our log B estimate uses Savage-Dickey KDE — it approximates but does not
   replace nested sampling. Treat our SD log B as ±1 at minimum.
2. Planck 2018 plik vs PR4 NPIPE: the NPIPE likelihood has 10-15% lower noise
   in temperature at ℓ > 1000. If Wolf+2025's log B came primarily from the
   CamSpec ℓ > 1000 region, our plik run may give a modestly lower log B.
   This is not a contradiction — it is a dataset difference.
3. The Cassini wall changes the effective prior volume: with the wall ON, the
   prior density at ξ_χ = 0 is unchanged but the prior domain shrinks, which
   mechanically raises the Savage-Dickey numerator. Account for this when comparing.
4. Wolf+2025 did not include KiDS-1000. Our levier1B run's added S₈ constraint
   independently tests whether the NMC solution remains viable under growth-sector
   pressure. This is the main novelty of levier1B vs Wolf+2025.
