WHAT THE PAPER CLAIMS (new in this version)
=======================================================================

PREDICTION. The phonon second-order coherence function near the sonic horizon
decays exponentially:

    g^(2)(tau) - 1 ~ exp(-Gamma * tau),

with decay rate

    Gamma = lambda_L^sonic = 2*pi * k_B * T_H / hbar.

For Steinhauer's 2019 measured T_H = 0.35 +/- 0.10 nK (arXiv:1809.00913),
this gives:

    Gamma = 287.9 +/- 82.3 s^-1,
    Krylov saturation time t_K = 1/Gamma = 3.47 ms.

This is NOT measured in Steinhauer 2019, which confirmed thermality via
momentum-space correlations, not the temporal g^(2)(tau) decay rate.

SIGNAL-TO-NOISE (new in this version, verified with sympy).
- Phonon pair emission rate: f_phonon = k_B * T_H / h = 7.3 Hz (single mode)
- For N_pairs = 10^4: integration time t_total = N_pairs / f_phonon = 23 min
- Statistical precision: sigma_Gamma / Gamma = 1/sqrt(N_pairs) = 1.0%
  (formal SNR = 100 sigma; likelihood ratio test trivially satisfied at N >= 25)
- Dominant systematic: T_H uncertainty (29%) propagates to 29% uncertainty on Gamma
- Coherence window: t_coh ~ 100 ms >> t_K = 3.47 ms (29 e-folds available — excellent)

THREE-PHASE EXPERIMENTAL PROGRAM (new in this version).
  Phase 1: Re-analyze Steinhauer 2019 raw time-series data for temporal g^(2)(tau)
           (no new data required; Technion group collaboration).
  Phase 2: Dedicated 23-minute measurement runs at Gamma_predicted = 288 s^-1.
  Phase 3: Vary BEC parameters (T_H, c_s) to test lambda_L = 2*pi*k_B*T_H/hbar
           as a scaling law over multiple parameter regimes.

=======================================================================
IMPORTANT CAVEATS (stated explicitly)
=======================================================================

1. KD THEOREM 4 DISCREPANCY. The Krylov-Diameter conjecture R_proper = xi predicts
   T_H ~ 1.2 nK vs. measured 0.35 nK (factor 3.5x). This is an order-of-magnitude
   prediction only. The actual surface gravity kappa = (1/2)|d(v-c_s)/dx|_H depends
   on the horizon flow profile, not just xi. ECI's precision prediction is the SCALING
   LAW lambda_L = 2*pi*k_B*T_H/hbar; the absolute value of T_H requires the
   Steinhauer BEC profile calculation.

2. MSS BOUND CAVEAT. Saturation of lambda_L = 2*pi*k_B*T_H/hbar is a prediction of
   the type II_inf / modular Lyapunov saturation framework. The BEC phonon system has
   no known gravity dual at finite xi. We state this as a conjecture to be tested,
   not as a derived theorem.

3. T_H UNCERTAINTY. At 29% uncertainty, the current T_H measurement limits the
   precision of the ECI test. A precision test requires T_H measured to <5%.
